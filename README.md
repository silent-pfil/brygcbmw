# Continuous Color Sequence

Black Red Yellow Green Cyan Blue Magenta White
repeating, of course.

**A simple request of a human to machine.**



Don’t try to make it look like something else.  

Just start doing something… and see what happens.



### What is this?



An interactive, endlessly evolving color pattern.  

Start with tiny repeating blocks of black → red → yellow → green → cyan → blue → magenta → white.  



As you increase the repeat size, the colors wrap continuously across rows like text in a document. What begins as neat stripes slowly transforms into shifting bands, chaotic interference patterns, hypnotic waves, and eventually massive solid fields.



It’s strangely meditative and addictive.



### Controls



- **Arrow Keys** (← ↓ → ↑) — manual adjustment (hold for fast)

- **Play Auto** — slowly increases the repeat value automatically

- **Slow (2s)** / **Fast (0.5s)** — change auto speed

- **Save PNG** — capture the current pattern

- Click/tap the canvas area first to enable keyboard control



### The Final Level



Keep going long enough and the entire screen becomes one solid color — usually pure black.  

The journey there is the real experience.



### Why it exists



This started as an experiment in emergent visual patterns. No grand plan, no deep meaning — just pure curiosity about what happens when a simple repeating rule meets continuous wrapping and human interaction.



Sometimes the most interesting things appear when you don’t force them to represent anything.



### How to run



1. Download `index.html` (or copy the code)

2. Open the file in any modern browser (no server needed)

3. Play. Watch. Lose track of time.
